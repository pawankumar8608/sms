# sms
Open Source Web Application based on Spring MVC, Hibernate, Spring Security, JSP, Mysql with XML Configuration

Prerequisites:-

JDK 1.7 or later 
Maven 3.3 or later 
MySQL 5.0 or later

Stack:-

Spring MVC4 
Spring Security 
Hibernate
Maven 
JSP 
MySQL
apache-tomcat-8.0.23 


Run:-

mvn clean install
